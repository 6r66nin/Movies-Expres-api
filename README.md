# Movies API

A simple REST API for managing movies and genres. Built with Express.js, TypeScript and PostgreSQL.

This is a small learning project. It is intended for practice and learning purposes.

## What is this?

A basic backend where you can:

- Create and manage movies
- Assign genres to movies
- Filter movies by genres
- Retrieve movies with all their genres

## Features

- CRUD operations for movies
- Filter movies by one or multiple genres
- Pagination support
- Data validation with Zod
- Error handling
- TypeScript support

## Tech Stack

- Node.js
- Express.js
- TypeScript
- PostgreSQL
- pg
- Zod
